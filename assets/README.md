# Archivos gráficos del juego

Sube aquí tus PNG/imágenes con **exactamente estos nombres** y el juego los usará solo.
Mientras no estén, el juego muestra marcadores y sigue siendo jugable.

| Archivo | Para qué | Formato |
|---|---|---|
| `novio.png` | Portada (zoom latido) y Puzzle (Prueba 1) | PNG cuadrado, recomendable 800×800 |
| `cigarro.png` | Prueba 2 (cigarro real estirable) | PNG horizontal, fondo recortado |
| `pollo1.png` … `pollo6.png` | Prueba 3 (sexador de pollos, de uno en uno) | PNG cuadrado |
| `novia.png` | Prueba 4 (ficha de la novia en el laberinto) | PNG con fondo transparente |
| `suegro.png` | Prueba 4 (meta: el suegro) | PNG con fondo transparente |
| `cancion.mp3` | Final, tras la cuenta atrás | MP3 (o cambia el nombre en game.js) |

> Para cambiar respuestas, márgenes o frases del Chivi, edita el bloque `CONFIG` al inicio de `game.js`.
